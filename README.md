# 📖 Project Overview

The **FIFA World Cup 2022 Player Performance Analysis** project explores player and team performance using data analytics techniques. The objective is to transform raw football statistics into meaningful insights through data cleaning, descriptive analysis, exploratory data analysis (EDA), trend analysis, and professional visualizations.

The project evaluates player performance using key metrics such as **Offensive Value**, **Defensive Value**, **VAEP Value**, **Minutes Played**, and **Expected Goals per 90 Minutes (xG per90)**.

This project demonstrates practical Data Analytics skills using Python and serves as a portfolio project for Data Analyst roles.

---

# 🎯 Business Problem

Football teams generate large amounts of player performance data during tournaments. However, raw statistics alone do not provide meaningful insights.

The goal of this project is to answer questions such as:

- Which team performed best offensively?
- Which team had the strongest defense?
- Who were the top-performing players?
- Which positions contributed the most?
- Is there any relationship between playing time and expected goals?
- Which players created the highest overall impact?
